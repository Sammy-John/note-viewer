Note Viewer App

Lightweight, Offline Markdown Viewer

A self-hosted, offline markdown viewer built to replace bloated note apps. I built it after Evernote’s free plan limited me to 50 notes — I needed a lightweight, portable system for study notes, project docs, and technical references.

🔧 Features

  • 🗂️ Dynamic folder-based sidebar
  • 📄 Markdown rendering with clipboard copy
  • 📅 Metadata support and date-based sorting
  • 🛠️ Fully offline and self-hosted
  • 💻 No server required — just open in browser

🧰 Tech Stack

  • HTML / CSS / JavaScript
  • Node.js (for generating the HTML structure)
  • Marked.js (Markdown parsing)
  • Vanilla JS for UI + clipboard

Get Started

Clone the repository and run it locally in minutes.

git clone https://github.com/yourusername/note-viewer.git
cd note-viewer
node src/generate_app.js
open index.html
